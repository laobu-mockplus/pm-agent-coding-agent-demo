## 根因

真实 LLM 调用返回失败：

- OpenAI：连接 `api.openai.com:443` 超时，错误码 `UND_ERR_CONNECT_TIMEOUT`。
- Azure OpenAI：endpoint 可达，但返回 `401`，提示 subscription key 或 endpoint 不正确。

前端原本把动作失败写进 `apiError`，但 `refreshAgentBus()` 每秒成功读取状态后会把 `apiError` 清空。因此用户点击后即使真实错误出现，也很快消失，看起来像按钮没有反应。
